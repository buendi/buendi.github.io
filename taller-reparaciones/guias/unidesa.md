## 📌 Problemas Comunes

### 1. Botones sin respuesta
**Síntomas:** Botones no registran pulsaciones, máquina no responde

**Pasos de diagnóstico:**
- [ ] Limpiar matriz de botones
- [ ] Revisar conector de botones
- [ ] Comprobar circuito control de entrada
- [ ] Verificar software

**Solución:**
```
1. Acceder a panel de control
2. Desconectar matriz de botones
3. Limpiar con aire comprimido
4. Reconectar y probar
5. Si persiste, revisar software
```

### 2. Sin sonido en altavoces
**Síntomas:** Sin audio, sonido bajo o distorsionado

**Pasos de diagnóstico:**
- [ ] Verificar volumen en software
- [ ] Revisar conexiones de altavoz
- [ ] Comprobar tarjeta de sonido
- [ ] Limpiar conectores
