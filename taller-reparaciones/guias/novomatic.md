## 📌 Problemas Comunes

### 1. La máquina no enciende
**Síntomas:** Pantalla negra, sin luces indicadoras

**Pasos de diagnóstico:**
- [ ] Verificar conexión de energía
- [ ] Revisar fusibles
- [ ] Comprobar cable de alimentación
- [ ] Inspeccionar batería de respaldo (si aplica)

**Solución:**
```
1. Desconectar la máquina
2. Esperar 30 segundos
3. Reconectar y reiniciar
4. Si persiste, revisar la fuente de poder
```

### 2. Pantalla parpadeante o sin imagen
**Síntomas:** Pantalla intermitente, colores extraños

**Pasos de diagnóstico:**
- [ ] Limpiar conectores de pantalla
- [ ] Revisar cable de video
- [ ] Comprobar RAM
